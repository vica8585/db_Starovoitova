Definition

HTML (hypertext markup language) is a description language used to structure documents on the web.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

History

Sir Timothy John Berners-Lee is an English computer scientist best known as the inventor of the World Wide Web.

Other technologies

Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Facts

In HTML 5, what was previously called block-level content is now called flow content.
